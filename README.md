# DoenerHub – Programmiertechnik Projektarbeit

## 1. Projektbeschreibung
DoenerHub ist eine Java-Swing-Anwendung zur Verwaltung von Döner-Bestellungen.
Der Benutzer kann über eine grafische Oberfläche Bestellungen erfassen, anzeigen
und nach bestimmten Kriterien filtern.

## 2. Funktionen
- Erstellen von Döner-Bestellungen über die GUI
- Preisberechnung basierend auf Toppings und Anzahl
- Anzeige von drei zufälligen TooGoodToGo-Bestellungen beim Start
- Filterung nach glutenfreien Bestellungen
- Fehlerbehandlung bei falschen Benutzereingaben

## 3. Klassenübersicht
- **DoenerHub**: GUI und Programmsteuerung
- **DoenerOrder**: Modellklasse für eine Bestellung
- **DoenerPrice**: Enum für Preisdefinitionen

## 4. Verwendete Technologien
- Java 17
- Java Swing
- Maven
- JUnit 5

## 5. JUnit-Test
Die Methode `getDetails()` der Klasse `DoenerOrder` wird mit JUnit getestet,
um sicherzustellen, dass eine gültige Textausgabe erzeugt wird.

## 6. Start des Programms
Starten Sie die Anwendung über die `main`-Methode der Klasse `DoenerHub`.
